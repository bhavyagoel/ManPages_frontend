Introduction

ManPages is a real-life implementation of the concept of page replacement in an operating system, where retailers will be making decisions on adding and removing products from their inventories following such algorithms. The efficiency of each inventory management strategy will be judged on the basis of “hits” & “misses” as per the page replacement algorithm. The most effective algorithm is then decided, by which algorithm can have the highest number of “Hit” ratio.

Algorithms

Following signifies the core essence of our project, the paging algorithms we have used.

First in First Out

FIFO is an abbreviation for first in, first out. It is a method for handling data structures where the first element is processed first and the newest element is processed last.

Optimal

In this algorithm, OS replaces the page that will not be used for the longest period of time in future.

Least Recently Used

Least Recently Used (LRU) algorithm is a Greedy algorithm where the page to be replaced is least recently used. The idea is based on locality of reference, the least recently used page is not likely
